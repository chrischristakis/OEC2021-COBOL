# OEC2021-COBOL
Team COBOL's entry for OEC 2021

## How to run:
First, copy all of the code from github by downloading it as a zip to your computer and putting it into a workspace.
Make sure all of the jar files in the lib folder are added to the build path of the project by selecting
Properties -> Build Path -> Add External Jars
And select each jar in the lib folder.
Then simply run the code and look at the console.
